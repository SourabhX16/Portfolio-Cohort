# HTML Resume Page

A single-page resume website built with pure HTML as part of the Web Dev Cohort 2026 assignment.

## Live Demo

🔗 **[View Live Resume](https://SourabhX16.github.io/Portfolio-Cohort/portfolio.html)**


## Setup Steps

1. Clone or download this repository to your local machine
2. No installation required - this is a pure HTML project with no dependencies
3. Simply open the `portfolio.html` file in any modern web browser (Chrome, Firefox, Safari, Edge, etc.)

## Usage

- Open `portfolio.html` directly in your browser by double-clicking the file, or
- Right-click on `portfolio.html` and select "Open with" → your preferred browser
- For a better development experience, you can use a local server:
  - Using Python: `python -m http.server 8000` (then visit http://localhost:8000)
  - Using VS Code: Install "Live Server" extension and click "Go Live"


## Technologies Used

- No CSS or JavaScript required
